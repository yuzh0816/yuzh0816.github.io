Hexo是一款快速、简洁且高效的博客框架

环境配置

Windows 篇

安装nodejs

进入nodejs中文网下载nodejs，32位64位安装包自选

安装git

进入git官网下载git，直接下载Windows包

Debian Linux篇

安装nodejs

同理，进入nodejs中文网下载nodejs,根据环境选择安装源码

我这里是树莓派4B，选择ARM v7，复制下载链接。

然后解压

进入目录并改名

安装Hexo
